# Run Frontend Dev Server

Start the Life Admin Agent React frontend using Vite.

## Steps

1. Navigate to the frontend directory and install dependencies if needed:
```
cd "/Users/rahulmalik/Desktop/Life Admin Agent/frontend"
npm install
```

// turbo
2. Start the Vite dev server:
```
cd "/Users/rahulmalik/Desktop/Life Admin Agent/frontend" && npm run dev
```

The frontend will be available at: http://localhost:5173

> **Note:** Make sure the backend is running at http://localhost:8000 before starting the frontend.
